### ブレストで合意したこと

| 領域 | 内容 |
|------|------|
| コンセプト | 家計可視化が主、節約は副。**コーチ機能は入れない** |
| Phase 10 | 複数 AI / API — 解析失敗（quota）の **根本対策** |
| Phase 11 | 月次予算、通知整理、分析タブ（棒全体 = 予算、必須/裁量積み上げ） |
| 予算 | オンボーディング③（APIキー直後）、スキップで無効、**新規のみ** |
| 集計 | デフォルト支出合計、設定で裁量のみに切替 |
| 通知 | マスター + 個別トグル。権限許可時は **失敗のみ ON** |
| 予算通知 | **10日・20日・月末** 固定 |
| カスタム性 | A（意味ある設定）のみ積極。テーマ色・通知日カスタムはやらない |
